基础#

三要素#

  1. 数据结构 = 逻辑结构 + 数据运算 + 物理结构
    • 逻辑结构=集合+线性+树形+图状
    • 运算=增删改查
    • 物理结构=存储结构=顺序+链式+索引+散列 存储
  2. 时间复杂度排序：O(1) < O(log~2~n) < O(n) < O(nlog~2~n) < O(n^2^) < O(n^3^) < O(2^n^) < O(n!) < O(n^n^)
    • 最坏&平均时间复杂度度量
    • 最深层循环的循环次数与n的关系
    • 乘法规则：相乘后度量
  3. 空间复杂度 内存中的变量存储大小

递归&动态规划#

递归：选定递归基 + 认定该方法可以解决该问题

  • 空间复杂度较高

动态规划：部分或完全消除算法中的递归成分（用有限空间记录子问题的解 以避免重复访问子问题） 条件：

  • 最优子结构（可能适用贪心）
  • 无后效性：已求解的子问题不受后续决策影响
  • 子问题重叠

队列#

先进先出 (first in first out,FIFO)

  1. 数组模拟队列 标记队列首尾(ql,qr)
    • 删除元素：ql++
    • 清空队列：ql=1,qr=0(ql>qr即可)
  2. 双栈模拟队列 F为队尾栈 S为队首栈
    • pop时若S为空 则将F元素弹出后插入S 再进行pop
  3. 双端队列 队首队尾均可插入删除元素
    • Python中容器为collections.deque
  4. 循环队列 下标为0的位置视为最后位的后继
    • 队列向数组尾部移动 最终入队即使有空位也会溢出（假溢出）

单调栈与单调队列#

保证元素单调性的栈/队列（递增or递减）
维护一个单增/单减的数组 实现最小/最大值查找\

  • (以单减找最大为例) 新元素与队尾元素比较 若更小则进队
  • 若更大 则从队尾开始将所有比之更小的元素踹出队 lc.239滑动窗口找最大\

int i=0;

for(;i<k;i++){

链表#

增删 O(1) 查读 O(n)

  1. 单向链表 指针 + 数据
  2. 双向链表 指针（左、右） + 数据
struct Node{
   int value;
   Node *left;
   Node *right;
};

插入数据#

  1. 单向链表
    • new node的next指向p的next node
    • p的next node指向new node 
    node->value=i;
node->next=p->next;
p->next=node;
  2. 单向循环链表 首尾相连 插入时判断是否为空
    • 若为空 new node的next指向自己 p指向new node 
    if(p==NULL){
  p=node;
  node->next=node;
}
  3. 双向循环链表 判断是否为空 & 修改左右两个指针 非空
    node->left = p;
node->right = p->right;
p->right->left = node;
p->right = node;

删除数据#

  1. 单向循环链表 删除p = p->next的值给p & 跳过p->next
    p->value = p->next->value;
Node *t = p->next;
p->next = p->next->next;
  2. 双向循环链表
    p->left->right = p->right;
p->right->left = p->left;
Node *t = p;
p = p->right;
delete t;

哈希表#

key->哈希函数->索引≈内存位置->value

  1. 哈希冲突 = 不同key计算的索引相同
    • 拉链法（开散列法）多键值索引到同一位置 在该位置建立链表
    • 闭散列法 所有记录记录在散列表中 发生冲突则根据某方式继续探查

堆#

树 每个节点的键值都大于等于/小于root节点的键值
大于等于的为最小化堆 反之为最大化堆（左子大于右子）

二叉堆#

  1. 结构：完全二叉树（以下以最大化堆为例）
  2. 插入：最下层最右叶子插入 向上调整
    • 复杂度O(logn)
  3. 删除（root）：root与最后的节点交换 向下调整
    • 向下调整 = 子节点中最大的与新root交换
    • 复杂度O(logn)
  4. 实现：在序列中 Hi的子节点为H2i与H2i+1
    void up(int x) {
  while (x > 1 && h[x] > h[x / 2]) {
   std::swap(h[x], h[x / 2]);
   x /= 2;
  }
}
void down(int x) {
  while (x * 2 <= n) {
   t = x * 2;
   if (t + 1 <= n && h[t + 1] > h[t]) t++;
   if (h[t] <= h[x]) break;
   std::swap(h[x], h[t]);
   x = t;
  }
}
  5. 建堆：向上/向下调整
    • 向上调整 从叶子开始向堆上部移动 复杂度O(nlogn)
    • 向下调整 从root开始向堆下部移动 复杂度O(n)

二叉树#

二叉搜索树#

左子<root<右子

删除元素：若有两个非空子节点 用左子树的最大值/右子树的最小值代替它后删除

平衡树#

每个节点的左子树和右子树高度差最多为1

  1. 调整过程
  • 右旋 root左子成为新root root成为新root的右子 新root原右子成为root左子 
    TreeNode* newRoot = root->left;
root->left = newRoot->right;
newRoot->right = root;
  1. 四种平衡破坏情况 LL RR省略
  • LR 左子的右子树过长
  • RL 右子的左子树过长（类上）

AVL树#

平衡的二叉搜索树
性质：

  • AVL的左右子树也是AVL树 且左右子树高度相差不大于1（平衡因子 = 右子树高 - 左子树高）
  • 树高logn

调整：若平衡因子=-2 == 左子左旋后root右旋

B树#

每个节点可以拥有两个以上的子节点 增删改查在对数时间完成
最下层的节点称为叶子节点 性质：（m阶）

  • 每个节点最多有m个子节点 非叶子节点（除root）最少有m/2个子节点
  • 若root不是叶子节点 则至少有2个子节点

排序#

冒泡排序#

对序列逐个扫描 n次扫描后排序完毕
最坏&平均情况下时间复杂度O(n^2^) 最好情况下时间复杂度O(n)

  bool flag = true;
  while (flag) {
    flag = false;
    for (int i = 1; i < n; ++i) {
      if (a[i] > a[i + 1]) {
        flag = true;
        int t = a[i];
        a[i] = a[i + 1];
        a[i + 1] = t;
      }
    }
  }

插入排序#

最坏&平均情况下时间复杂度O(n^2^) 最好情况下时间复杂度O(n)\

  • 从头开始遍历每个需插入的数据
  • 与其前已排序的数据的末尾向头逐一比较
for (int i = 1; i < len; ++i) {
    int key = arr[i];
    int j = i - 1;
    while (j >= 0 && arr[j] > key) {
      arr[j + 1] = arr[j];
      j--;
    }
    arr[j + 1] = key;
  }
  • 折半插入 二分算法优化 时间复杂度不变

快速排序#

将数列划分为2parts 递归到两子序列中进行快排 = 挖坑填数
最优/平均时间复杂度O(nlogn) 最坏O(n^2^)

  • 若选第一个数为基准数 数组首尾两指针 i&j

  • 从j开始向前找小于或等于基准数的 放到原基准数的位置

  • 从i开始向后找大于基准数的 放到原j的位置

  • 以此循环 最后i=j放入基准数

  • 在前后两parts再继续进行以上步骤

  • 三数取中：前中后三个数先排序——中与后换（后的前一个也可以）——从 前 到 后的前一个 进行大小判断（前指针先动）——后和指针位置处数字交换——重复

    • 排序后再选pivot
    • while内每一步也要限定i<j或i<=j（且放在&&号之前）
void quick_sort(int s[],int l,int r){
   if(l<r)
   {
      int i=l,j=r,base=s[l];
      while(i<j){
        while(i<j&&s[j]>base)
            j--;
        if(i<j){
            s[i++]=s[j];
        }
        while(i<j&&s[i]<=base)
            i++;
        if(i<j){
            s[j--]=s[i];
        }
      }
   }
   s[i]=base;
   quick_sort(s,l,i-1);
   quick_sort(s,i+1,r);
}

优化：

  • 三路快速排序
    • 三路取中（首、中、尾三元素进行比较后选择base）
    • 与base相等的元素聚集在其周边
  • 短序列用插排

归并排序#

时间复杂度O(nlogn) 空间复杂度O(n) 类似快排，但快排将数组分为大小两半 省略了最后将两部分数组合二为一的步骤

堆排序#

二叉堆 = 完全二叉树（/近似） 最大堆/最小堆
时间复杂度：$O(nlogn)$ 空间复杂度：$log(n)$ 递归使用栈的空间代价

  1. 拥有一个arr
  2. 对arr建最大堆——只检查倒数第二行及以上，使用heapify
  3. 循环检查 从最后一个数字检查到第二行（即不包括root）——将root与当前数字交换后，从顶向下heapify 过程中可以将len–（后面的已经检查过了）
  4. heapify：左子右子定义——检查parent是否比左子右子大——否 则将其与最大值交换后向下继续heapify（更新被交换的子的子树）
   public static void main(String[] args) {
        int[] arr={6,3,8,2,9,4,7,1,5};
        int len=arr.length;
        buildMaxHeap(arr,len);
        for(int i=len-1;i>0;i--){
            swap(arr,0,i);
            len--;
            heapify(arr,0,len);
        }
        System.out.println(Arrays.toString(arr));
    }
    public static void buildMaxHeap(int[] arr,int len) {
        for(int i=len/2;i>=0;i--){
            heapify(arr,i,len);
        }
    }
    public static void heapify(int[] arr,int i,int len) {
        int left=i*2+1;
        int right=i*2+2;
        int largest=i;
        if(left<len && arr[left]>arr[largest]) largest=left;
        if(right<len&& arr[right]>arr[largest]) largest=right;
        if(largest!=i){
            swap(arr,largest,i);
            heapify(arr,largest,len);
        }
    }
    public static void swap(int[] arr, int i, int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }

计数排序#

  1. 原理：确定数组中max min，所有元素排入count[value-min]中，相同元素则count加一。
    • 插入完毕，count[n]+=count[n-1]——为了确定在新数组中的位置
    • 从后向前遍历原数组（相同元素插入时从前向后，取出时从后向前符合index）
    • 在新数组中的count[n]-1位置（数组从0开始但count最小位为1）放入arr[i]，count[n]减一

基数排序#

  1. 原理：从每个数的个位到十位到最高位依次排序，排进0-9的队列中，一直到max的最高位

计数&基数&桶#

  1. 计数：每个桶放单一键值
  2. 基数：根据键值的每位数字分配桶（桶的映射函数方法不同）
  3. 桶：每个桶存储一定范围的数值
