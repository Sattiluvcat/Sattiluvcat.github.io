基本#

Java8新特性，作为Future的增强版本，提供灵活的异步编程能力

核心任务：线程池异步执行任务 –> 每个操作生成新CF对象，共同组成任务链

具体操作：应用回调函数BiConsumer（andThen()方法）/Function支持执行后续任务 & 通过volatile变量与CAS进行状态维护

应用#

创建#

在指定线程池中创建：

ExecutorService executor = Executors.newFixedThreadPool(2);
CompletableFuture.supplyAsync(() -> "Task", executor);

若不指定线程池，会在其默认线程池ForkJoinPool.commonPool()中执行，不推荐

链式处理#

对结果的三种处理方式：

  • 消费：thenAccept()

      CompletableFuture.supplyAsync(() -> "Hello").thenAccept(s -> System.out.println(s)); // 输出 "Hello"

  • 转换：thenApply()

      CompletableFuture<String> future = CompletableFuture.supplyAsync(() -> "Hello").thenApply(s -> s + " World"); // 输出 "Hello World"

  • 组合：thenCompose()

    thenCompose()是将任务A的执行结果作为任务B的输入

      CompletableFuture<String> future = CompletableFuture.supplyAsync(() -> "Hello").thenCompose(s -> CompletableFuture.supplyAsync(() -> s + " World"));

组合处理#

任务间不相互依赖

  • 合并结果：thenCombine()

    thenCombine()是等待任务A、任务B的结果并共同处理

      CompletableFuture<String> cf1 = CompletableFuture.supplyAsync(() -> "Hello");
  CompletableFuture<String> cf2 = CompletableFuture.supplyAsync(() -> "World");
  cf1.thenCombine(cf2, (s1, s2) -> s1 + " " + s2); // "Hello World"

  • 等待所有任务完成：allOf()

      CompletableFuture<Void> all = CompletableFuture.allOf(cf1, cf2);
  all.thenRun(() -> System.out.println("All tasks done"));

  • 等待任一任务完成：anyOf()

      CompletableFuture<Object> any = CompletableFuture.anyOf(cf1, cf2);
  any.thenAccept(result -> System.out.println("First result: " + result));

异常处理#

  • 捕获异常并处理
      CompletableFuture.supplyAsync(() -> {
      if (error) throw new RuntimeException("Error");
      return "OK";
  }).exceptionally(ex -> "Fallback"); // 返回默认值
    可以用completableFuture.get()或completableFuture.join()阻塞当前线程（等待）获取异步任务结果，二者区别如下：
    • get()：抛出受检异常（ExecutionException，需显式捕获，如try-catch） & （如调用Thread.interrupt()时）线程中断会抛出InterruptedException
    • join()：抛出非受检异常（CompletionException完成异常，任务运行时出现的异常被包装为该类），适用链式调用 & 不会响应线程中断（当前线程被设置为interrupted状态也不会停止阻塞，直到异步任务完成）

超时处理#

Java 9后添加 使用orTimeout与completeOnTimeout设定超时时间并处理

CompletableFuture.supplyAsync(() -> "Task")
    .orTimeout(1, TimeUnit.SECONDS) // 超时抛出 TimeoutException
    .completeOnTimeout("Default", 1, TimeUnit.SECONDS); // 超时返回默认值

实例#

主页上显示用户数据与商品排行榜

controller层设置#

@RestController
@RequestMapping("/user")
public class HomeController {

    @Autowired
    private HomeService homeService;

    @GetMapping("/main")
    public CompletableFuture<ResponseEntity<HomeData>> getHomePage() {
        return homeService.loadHomeDataAsync()
            .thenApply(ResponseEntity::ok) // 成功时返回 200 OK + HomeData
            .exceptionally(ex -> {
                return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).build();
            });
    }
}

由Spring等待CompletableFuture执行完成并处理，返回到前端的就是ResponseEntity<HomeData>对象

data设置#

HomeData只设置两个信息：用户数据与排行榜，可按需设置其它数据

@Data // Lombok 注解，自动生成 getter/setter
public class HomeData {
    private Integer userCount;
    private List<ProductRanking> productRankings;
    // 其他字段……
}

service层实现#

@Service
public class HomeServiceImpl implements HomeService {

    @Autowired
    private UserMapper userMapper;  // 用户数据库访问
    
    @Autowired
    private ProductMapper productMapper;    // 商品数据库

    @Override
    public CompletableFuture<HomeData> loadHomeDataAsync() {
        // 自定义线程池-->可设置为Bean直接注入
        Executor asyncExecutor = Executors.newFixedThreadPool(4);

        // 并行执行两个异步任务
        CompletableFuture<Integer> userCountFuture = CompletableFuture.supplyAsync(
            () -> userMapper.getUserCount(), // 获取用户数据
            asyncExecutor
        );

        CompletableFuture<List<ProductRanking>> productRankingFuture = CompletableFuture.supplyAsync(
            () -> productMapper.getProductRankings(), // 获取排行榜
            asyncExecutor
        );

        // 合并结果，若有多个结果需要合并，可以使用 allOf()
        return userCountFuture.thenCombine(productRankingFuture, (userCount, rankings) -> {
            HomeData homeData = new HomeData();
            homeData.setUserCount(userCount);
            homeData.setProductRankings(rankings);
            return homeData;
        }).exceptionally(ex -> { // 异常处理
            log.error("加载首页数据失败", ex);
            return new HomeData(); // 返回空对象或默认值
        });
    }
}
